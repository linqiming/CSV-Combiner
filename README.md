# CSV-Combiner

Solution of PMG csv-combiner challenge. A command line program that take several CSV files to a new CSV file. 

##  Requirements
* Python 3.11.1
* pandas
```
$ pip install -r requirements.txt
```

## Usage

### csvcombiner.py

```
$ python3 ./csvcombiner.py ./fixtures/accessories.csv ./fixtures/clothing.csv > combined.csv 
```

### testcsvcombiner.py
```
$ python3 ./testcsvcombiner.py
```
